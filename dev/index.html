BandedMats.inv_diag — Function

retrieve the diagonal of the inverse of a banded and symmetric matrix B

一种带状对称矩阵的快速算法，用于Whittaker smoother求解。注意B需要是对称。

根据U矩阵，计算B^-1的对角线元素。

为配合LDL'分解，这里输入变量采用的是L。

\[B = (U' * D * U) # Hutchinson 1985, Eq. 3.1 B^(-1) = B * U^(-1)' + (1 - U) * B^(-1) # Hutchinson 1985, Eq. 3.3\]

Arguments

  • U: [ 1 c₁ e₁ f₁ 0 0 1 c₂ e₂ f₂ 0 0 1 c₃ e₃ 0 0 0 1 c₄ 0 0 0 0 1

]

  • U2: [ c₁ e₁ f₁ c₂ e₂ f₂ c₃ e₃ 0 c₄ 0 0 0 0 0

]

Examples

S = inv_diag(L, d)

Dongdong Kong, CUG, 2024-05-07

References

  1. Hutchinson, Michael F., and Frank R. de Hoog. "Smoothing noisy data with spline functions." Numerische Mathematik 47 (1985): 99-106.
